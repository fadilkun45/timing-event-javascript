SetTimeout Events trigger

setTimeout adalah sebuah trigger pemanggil events dengan hitungan mundur. SetTimeout bisa di panggil secara otomatis atau pun dengan manual. untuk yang manual contoh nya menggunakan button dan yang otomatis langsung terpanggil saat user baru meng akses web.

SetInterval Events trigger

setInterval sama seperti SetTimeout yaitu untuk mentrigger event dengan waktu. namun beda nya kalo SetInterval digunakan untuk mengulang event tergantung berberapa detik si set interval di setting
